Thank you for downloading.

-Models folder contains MosasaurusModel.fbx.
Mosasaurusver1.0.FBX is the old version of mosasaurus model.
This will be deleted in later update.
-Models/Materials folder contains Textures.
-Animations folder contains FBX files of animations.
These are main contents of this package.

Followings are example use of this package.
-Scenes folder contains a demo scene.
-Scripts folder contains scripts that was used in the demo scene.
-Prefabs folder contains a prefab that was used in the demo scene.
-Animator folder contains an animator controller that was used in the prefab.
