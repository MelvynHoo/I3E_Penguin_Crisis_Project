Thank you for downloading Frog And Skeleton.

-Models folder contains original fbx file of frog.
Walk Animations are compatible with root motion.
-Animations folder contains additional animations.

-Prefabs folder contains 6 types of prefabs of frog.
These prefabs are attached rigidbody, collider, animator and scripts.

-Scenes folder contains demo Scenes that uses 3 types of prefabs of frog.

-Old folder contains model of old version.
